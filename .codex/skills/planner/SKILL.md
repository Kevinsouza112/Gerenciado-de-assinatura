# Planner

Before coding, define:
- Goal of the task
- User flow
- Required pages
- Required database changes
- Small implementation steps
- Risks and dependencies
- Acceptance criteria

Prefer small, incremental tasks.
Do not build the whole system at once.
For this project, always keep the MVP simple and focused on a restaurant digital menu and order management system.
