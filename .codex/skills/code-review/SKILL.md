# Code Review

Review code like a senior engineer.

## Analyze

- readability
- maintainability
- architecture
- duplication
- naming
- performance
- scalability
- security

## Identify

- code smells
- unnecessary complexity
- bad abstractions
- missing validations
- potential bugs

## Suggest

- cleaner implementations
- simpler logic
- better naming
- architectural improvements

## Review Style

Provide:

1. strengths
2. issues
3. improvements
4. refactored examples
