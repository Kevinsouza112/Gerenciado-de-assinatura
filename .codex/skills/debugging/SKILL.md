# Debugging Mode

Never guess.

Follow process:

1. Reproduce
2. Identify
3. Isolate
4. Analyze
5. Fix
6. Verify

## Always Check

- stack traces
- logs
- API responses
- async behavior
- environment variables
- database connection
- request payloads

## Debug Strategy

Prefer:

- root cause analysis
- minimal reproducible examples
- logging
- structured debugging

## Never

- blindly change code
- ignore exceptions
- remove validations without explanation
